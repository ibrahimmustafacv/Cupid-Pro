# Cupid-Pro - Advanced Targeted Password Generator

Cupid-Pro is an **interactive command-line tool** for generating custom password lists based on personal information. It is designed for **authorized security testing** and **password strength auditing** on your own systems.

## Features
- 🎯 **Three attack modes**: Social Media, WiFi, System, or Mix All.
- 🧠 **Intelligent patterns**: Combines names, years, numbers, pets, colors, etc.
- 🖥️ **Beautiful CLI**: Rich tables and progress bars.
- 📂 **Save output**: Export passwords to a file.
